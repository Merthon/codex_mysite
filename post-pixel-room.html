给自己搭一个像素房间

2026.05.16 · 创作 · 技术

我希望这个博客不是一张安静的纸，而像一台老游戏机：按钮有反馈，头像有颗粒感，页面滚动时能留下读到哪里的痕迹。

第一步：确定像素语言

像素风最重要的不是把所有东西都做成小方块，而是建立清晰的规则：硬边阴影、有限色彩、网格节奏和明确的交互状态。

第二步：让头像进入系统

站点直接使用现有的 my.jpg。通过 CSS 的 image-rendering: pixelated 和方形边框，让头像自然融入页面，同时也把它作为 favicon。

第三步：让静态页面动起来

搜索、筛选、主题切换、阅读进度、留言板都用原生 JavaScript 完成。没有框架也可以有足够的互动，只要功能贴近页面本身。
